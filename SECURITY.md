## Security Policy

### Reporting Vulnerabilities

- **Please report security vulnerabilities privately** to `security@kiwi.eu.com`.
- **Do not open public GitHub issues or pull requests** for security-sensitive findings.

### Scope

- TON Jetton smart contracts in this repository (for KWT).
- Deployment, configuration, metadata, and verification scripts used with these contracts.

### Handling of Reports

- Valid reports will be **acknowledged** as soon as reasonably possible.
- Issues will be **triaged and prioritized** based on impact and exploitability.
- Additional details may be requested to help reproduce and verify the issue.
